# iRacing Telemetry Analyzer & Replay Director

This project aims to provide tools for analyzing iRacing telemetry data and potentially directing replay footage based on interesting events.

## Features (Planned)

*   Telemetry data parsing and analysis
*   Event detection (incidents, overtakes, battles, etc.)
*   Replay control and automation
*   GUI for easy interaction

## Project Structure

*   `src/`: Contains the core application code.
*   `tests/`: Contains unit and integration tests.
*   `docs/`: Contains project documentation.
*   `pyproject.toml`: Defines project metadata, dependencies, and build system configuration.
